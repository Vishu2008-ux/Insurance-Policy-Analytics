# 🛡️ Insurance Policy Analytics Dashboard

## 📌 Project Overview
An interactive Insurance Dashboard built in Power BI using Power Query, DAX, data modeling, KPIs, slicers, and advanced visualizations to analyze policies, claims, customer demographics, payment status, and premium growth.

## 🛠️ Tools Used
- Power BI Desktop
- Power Query
- DAX
- Data Modeling
- Microsoft Excel

## 📊 Dashboard Features
- Total Policies, Customers & Claim Amount KPIs
- Age & Gender Analysis
- Policy Type Analysis
- Claim Status Analysis
- Payment Status Tracking
- Premium Growth Trend
- Policy Expiry Analysis
- Interactive Year & Status Slicers
